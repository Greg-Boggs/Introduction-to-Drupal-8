What is a static website?

Static Website Generator

A static generator does the building when you create your content. Write once. Read many times. Learn More >

A generator you may have heard of is called Jekyll. I like Hugo. It's like WordPress. Except it's fast, won't be hacked, and doesn't need security updates.

We seriously need to update Wikipedia.

Static Advantages

  • Very, very fast
  • Extremely secure
  • Efficient to develop with
  • Easy to learn
  • Easy to edit
  • Easy to host

Let's define the common pieces.

  • Markdown, Markdown, Markdown More >
  • Git version control
  • Metadata usually in Yaml. author: Greg Boggs
  • Plain HTML files, just like 1997, but without the capslock.
  • A template language like Twig, Mustache, or Slim*
  • Command line tools
  • Free, open-source software
